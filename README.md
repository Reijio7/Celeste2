# *Celeste 2*

A full-scale platformer inspired by Celeste — built as an expanded, modern reimagining of the PICO-8 version of *Celeste 2*.

## About the Game

**Celeste2** is a complete, standalone game that takes the original concept from the PICO-8 prototype and evolves it into a full experience similar in scope and depth to *Celeste*.

This is **not the PICO-8 version** — it's a full game with enhanced mechanics, visuals, and level design.

## Development Status

Currently in active development.

The game is being built step by step, starting from core systems and expanding toward a polished, complete experience.
